WHAT IS ANVIL?

When a PR opens, ANVIL takes over. Seven specialized agents fan out in parallel — reviewing code quality, scanning for secrets and injection patterns, generating and running real pytest suites against the PR branch. If tests fail, a Self-Healer attempts up to three automated fixes. A Decision Agent then issues the final verdict: merge, reject, or file a GitHub Issue. No human needed, ever.
