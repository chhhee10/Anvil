WHAT IS ANVIL?

ANVIL is a multi-agent autonomous pipeline that reviews your pull requests without human intervention. We combine five specialized AI agents — Orchestrator, Researcher, Code Analyst, Writer, and Critic — working in parallel to deliver comprehensive PR reviews in seconds, not hours. Each run is webhook-triggered, fully autonomous, and streamed live to your dashboard.
